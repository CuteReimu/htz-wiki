属性这个词有两种含义：
- 属性点：力量、魔力、体质、耐力、敏捷
- 属性结果：例如攻击、防御、速度、灵力、血量等

由于任何来源增加属性点都会直接加成属性结果，**对人物而言**，每一点属性点影响的属性结果如下：
- 力量：0.7攻击 + 0.4灵力 + 0.1速度
- 魔力：0.7灵力
- 体质：5血量（吃强身加成） + 0.1灵力 + 0.1速度
- 耐力：1.5防御 + 0.1灵力 + 0.1速度
- 敏捷：0.7速度

其中，1点灵力等价于1点法术攻击+1点法术防御，而有部分技能或者效果只加法术攻击或者只加法术防御，记得和灵力区分开。

通过以上数值，我们可以得出一些结论：
- 魔力不加任何速度，因此法系门派的速度偏低
- 敏捷不加任何灵力，体质、耐力加的灵力很少，因此辅助、封系的灵力偏低
- 1力量 = 0.7攻击，1耐力 = 1.5防御，说明耐力的价值远比力量高。**在玩家PK中**，也产生了一种思路，降攻提防，以提高生存能力。
