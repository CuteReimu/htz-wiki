## 宝石等级

设当前等级宝石等级为 $x$ ，则锻炼到下一级的宝石消耗为：

$$
\left\{
\begin{aligned}
&2^x &&\text{ 当 } x \le 4\\
&8x-16 &&\text{ 当 } 4 \le x \le 11\\
&72 \times 2^{x-11} &&\text{ 当 } x \ge 11
\end{aligned}
\right.
$$

## 宝石转移

同等级、同部位随便转移，如果有任意一件装备从未参与过转移，则免费，否则花费少量银币。

同部位，非同等级。保留装备上等级/10-1的宝石锻位，最多保留13级。\
例如，90级装备上的宝石转移到100级及以上装备时，可以保留8段。

## 宝石效果

- 红玛瑙，40血 &asymp; 8体
- 月亮石，12防 &asymp; 8耐
- 舍利子，6灵 &asymp; 8.57魔
- 黑宝石，8速 &asymp; 11.43敏
- 太阳石，8攻 &asymp; 11.43力
- 吸收宝石，对应属性2%吸收。

补充说明：
1. 体质[属性点](stat.md)和红玛瑙都可以被强身放大，所以在属性换算的时候，无需考虑强身。
2. 虽然都是属性，但不同属性对不同职业的意义也是不一样的。比如物理输出在能锻炼太阳石的部位一定会选择太阳石，法系输出对舍利子的需求同理。
3. 如果你需要速度，那么黑宝石收益非常高。
4. 易成长+20%的宝石效果，只保留整数部分。
5. 除法系输出门派以外，其他门派锻炼项链上的宝石时，优先选择吸收宝石，而非舍利子。（特殊战术除外）
