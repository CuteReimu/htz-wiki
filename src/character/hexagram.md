---
title: 卦象
---

可以在铁匠铺消耗信誉或现金对装备部位开启**卦象**，在帮派四象炉处开启**卦象**可以获得一定的减免。由于信誉和现金都非常容易获得，因此卦象对普通玩家来说是一个非常友好的系统。

**卦象**与装备本身不绑定，只与装备部位绑定，一共有6个部位，因此可以开启6个卦象。

每次开启卦象会有随机80~110天的有效期。

如果有频繁切换卦象的需求，可以在铁匠铺处开启第二套卦象。

卦象的效果分为**主效果**和**附加效果**。开启卦象时，主效果的类型和数值随机，当主效果的数值达到一定值时，会获得对应的附加效果。附加效果的类型和主效果的类型绑定，关系如下表所示。

| 主效果最大值             | 附加效果最大值                                                      |
|--------------------|--------------------------------------------------------------|
| 攻击+40              | 普通攻击击倒目标后25%随机追击下一个目标                                        |
| 防御+35              | **封印与抗封印成功率增加5%**                                            |
| 灵力+21              | 对敌方宠物增加装备等级/2+20的伤害结果                                        |
| 气血+110（吃强身加成）      | **使用特技消耗愤怒减少10%**，和[愤怒](/equipment_special/fen_nu.md)神通共存时无效 |
| 魔法+300             | 使用法术时有15%几率不消耗魔法                                             |
| 速度+22              | 获得增益状态时有5%几率回合数+1                                            |
| 受到的恢复效果增加+10%      | **增加愤怒时额外增加7%+1**                                            |
| 受到的伤害结果-32         | 受到暴击或狂暴的伤害结果减少20%                                            |
| 伤害结果+32            | 30%几率无视任何物理反击效果                                              |
| 受到异常状态10%使该状态回合数-1 | **恢复类法术或特技效果+20%**                                           |

**全身6条卦象中，同类型的主效果只能生效三条，同类型的附加效果只能生效一条。**

如果只看主属性，物理输出需要3条攻击，法术输出需要3条灵力，这是毋庸置疑的。

对于需要防御、血量、速度的辅助和封系门派而言，可以简单计算一下：
- 35防御 &asymp; 23.3耐力
- 110气血 &asymp; 22体质
- 22速度 &asymp; 31.4敏捷

很显然，**22速度 &gt; 35防御 &asymp; 110气血**。

::: tip 日常卦象选择

- 物理输出：3条攻击、3条速度
- 法系输出（适当抢速）：3条灵力、3条速度
- 法系输出（完全放弃速度）：3条灵力、剩下3条选择血量或防御
- 辅助和封系：3条速度、剩下3条选择血量或防御

**追求满值花费巨大，普通玩家完全没必要追求满值**，差几点属性影响不大。

:::
